# Instagram Automation Bots 🤖

Two 100% safe Python bots for Instagram creators and businesses. No login required. No API. No ban risk.

## 📂 What's Inside

### 1. /insta-hashtag-bot/
Generates 100+ viral, low-competition hashtags for any niche in 5 seconds.

**How to Run:**
```bash
cd insta-hashtag-bot
pip install -r ../requirements.txt 
python insta_hashtag_bot.py
```
### 2. /insta-dm-bot/
Generates 50 copy-paste DM reply templates for Price, Order, and FAQ.

**How to Run:**
```bash
cd insta-dm-bot
python insta_dm_bot.py
```
